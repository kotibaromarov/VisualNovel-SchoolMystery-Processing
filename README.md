# Visual Novel "The School Day"

[![Processing](https://img.shields.io/badge/Made%20with-Processing-0066CC?style=for-the-badge&logo=processing&logoColor=white)](https://processing.org/)

A short interactive visual novel with a branching narrative, custom UI, and original artwork. This is my first completed game project for university, developed from scratch in 7 days.

## Gameplay Demo

https://youtu.be/6A4iRhgEWwk

## Features

- **Branching Narrative:** Player choices affect the story and lead to different outcomes.
- **Custom Dialogue System:** Built with text files and a unique "keyword" path-tracking system.
- **Original Assets:** All character sprites and UI elements were created by me.
- **Interactive UI:** A fully functional main menu, settings menu (with sound toggles), and in-game choices.
- **Sound Design:** Custom sound effects for UI interaction.

## Purpose of the Project

This project was my first step into game development. I wanted to create a complete, playable experience to learn about game loops, player input handling, and data management. It demonstrates my ability to:

- **Problem-solve:** I designed a custom logic system to manage the branching paths and save the player's progress.
- **Learn Quickly:** I built this in just 7 days while learning the Processing language.
- **Finish What I Start:** The project is fully playable demo, from the main menu to the final outcome.

## Technologies Used

- **Language:** Processing (Java-based)
- **Libraries:** Processing Sound
- **Tools:** GitHub, Paint Tool SAI (for sprites)

## How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kotibaromarov/VisualNovel-SchoolMystery-Processing.git
    ```
2.  **Open in Processing:**

   - Launch the Processing IDE.
   - Click File -> Open... and select the novella.pde file.

4.  **Run the sketch:**

        Click the "Run" button (the play icon) in the top-left corner.

(You may need to install the Sound library via the Contribution Manager if it's not already present).
